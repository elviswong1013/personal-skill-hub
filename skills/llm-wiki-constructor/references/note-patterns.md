# Preferred Note Patterns

Consult this reference when creating pages during Step 6 (Promote Knowledge).

## Raw Note Pattern

- title, source metadata, excerpt or summary
- first-pass value judgment
- initial links or candidate promotions

## Source Note Pattern

- title and subtopic scope
- synthesis (agreements, disagreements, support levels)
- links to raw items drawn from
- candidate entities and concepts spotted

## Entity Note Pattern

- name and type (person, organization, tool, role, framework, book, platform, artifact)
- why it matters, key attributes or contributions
- related entities and concepts (as `[[wikilinks]]`)
- source pages that support this entity

## Concept Note Pattern

- definition, why it matters, key points
- relation to other concepts
- common confusions or limits

## Wiki Note Pattern

- summary, explanatory sections
- route guidance, cross-links to concept and entity notes
